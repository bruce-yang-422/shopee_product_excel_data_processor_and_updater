# data/input 說明

請將蝦皮後台下載的商品批次更新 Excel 放置於此資料夾。

## 檔案命名規則

```text
SH????_*商品資料*_??.xlsx
```

| 部分 | 說明 | 範例 |
| --- | --- | --- |
| `SH????` | 店家代碼（SH + 4 字元） | `SH0005` |
| `*商品資料*` | 任意文字，須包含「商品資料」 | `有才寵物商店_商品資料` |
| `_??` | 2 位數序號 | `_01` |

**範例檔名：** `SH0005_有才寵物商店_商品資料_01.xlsx`

> 不符合命名規則的檔案會被略過，並於執行記錄中顯示警告。
